# ⚡ Thinkly — DDCET Exam Preparation Platform

A smart exam prep platform for Gujarat Diploma students preparing for DDCET.

## Features
- 📚 Chapter-wise Practice Quizzes
- ⏱ Timed Mock Tests (Full Syllabus)
- 🧠 AI Tutor (Powered by Claude AI)
- 🏆 Leaderboard
- 📋 Syllabus Progress Tracker
- 📊 Result History & Analysis

## Tech Stack
- React 18 + Vite
- Claude AI API (AI Tutor)
- Deployed on Netlify

## Local Development

```bash
npm install
npm run dev
```

## Deploy to Netlify

1. Push to GitHub
2. Connect repo to Netlify
3. Build command: `npm run build`
4. Publish directory: `dist`

Or drag & drop the `dist` folder into Netlify after running `npm run build`.
